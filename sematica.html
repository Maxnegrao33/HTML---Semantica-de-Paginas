O Html tem que fazer sentido

Essa é a definição da Semântica

Sobre o Html5 XHTML x HTML5

Algumas diferenças do XHTML para p HTML5 são bem visíveis

Dentro do Body há algumas tags com significados

Os quais traremos os seus dignificados nos quadros abaixo:

Header

  • Cabeçalho da página

    • Aqui onde se concentra parte das informações da página principal

    Geralmente contém também a Barra de Menus para Navegação

    Página Principal

    Quem Somos

    Contato

    E sua ligação (link/ _blank) para as respectivas páginas

    Main

  • Definição do Corpo Principal

    • Aqui onde vai a parte principal do código

    As seções: Article, Nav, FigCaption,

    Esta tag não pode ser filha (herdeira de nenhuma)

    Ela precisa ser a tag principal, o conteúdo a ser buscado pelo bot

    Estas tags podem estar aqui dentro

  • Header, Footer, Article

    • Pode substituir o article por section

    Não pode haver mais de um elemento main no documento

    Aside

  • Campo ao lado do Corpo principal

    • Pode também conter elementos do tipo Menu

    Ou pequenos quadros de notícias ou colaboradores

    Pode estar em qualquer um dos lados (left/right), barras laterais

    Assunto sem ligação com o assunto principal da página

    A semântica dele é diferente da section, com menor importância

    Section

  • Divide o corpo da página em seções

    • O elemento section pode ser declarado dentro dos outros elementos

    Define uma seção sobre algum artigo ou notícia.

    Não tem limites para colocar a section, são seções de informações

    Lista de resultado de pesquisas, podem ser inseridos dentro de uma section

    Nav

  • Links de Navegação - Pode ter vários

    • No Footer - São Links de Rodapé

    Por padrão, utiliza uma lista não ordenada

    Pode ser utilizados dentros dos campos

    Article

  • Seção para escrever artigos

    • Esta seção pode ser usada em diversas outras partes da divisão da página

    Muito usada para expor notícias, imagens, links e outras

    Permite colocar um article dentro do outro, relacionando como uma herança

    Pode ser usado dentro do article para mostrar as horas

    Também pode ser usado

    Blockquote e Q

  • Blockquote pode ir dentro do Article, gerando uma referência sobre a citação -

    • Liga a citação que outra pessoa falou, geralmente uma url

    Q

  • Também faz uma citação

    • A citação "q cite="" " não quebra a linha

    Figure

  • Figure - Seção para inserção de figuras

    • É praticamente um container - um div para inserir coisas dentro dela

    Delimita o escopo da figura

    Figcaption

  • Figcaption - Vai dentro da Figure, como se fosse uma descrição da imagem

    • Podendo ser um rodapé com apenas algo sobre a imagem, ou um texto também

    Picture

  • Picture - É diferente do Figure

    • Pode colocar diversas imagens dentro do picture

    Sua sintaxe é "source srcset="car-image.jpg" " media="(min-width: 1000px)"

    Footer

  • Rodapé da página

    • É a parte final da página, onde estarão inseridas algumas informações

    Como a marca registrada, especificação de contatos, mapas, endereçoa e demais informações

    Um tag interessante é a tag address

    ARIA

  • Acessible Rich Internet Applications

    • Atributos inseridos no HTML - Árvore de acessibilidade

    Acessibility - Enable full page acessibility free

    Ex: ul class="menu" aria-hidden="true"

    Web Scraping

  • Crawler (Rastreadores - Robôs[aplicações])

    • Entra no site e lêem o HTML, após isso indexam.

    GoogleBot - Indexação das páginas Web no buscador do boot

    GoogleSearch

    Web Scraping minera e recolhe dados que estão expostos em plataformas de url públicas

    Interessante: Estudar sobre o schema.org (para motores de busca)

    SEO

  • Search Engine Optimize

    • Está de certa forma ligado com o Web Scraping

    É uma otimização para mecanismos de buscas

    É um conjunto de técnicas para melhorar as buscas do site do seu cliente

    Este SEO trabalha com as novas semânticas de Código (HTML5)

    Google Search Console, faz verificações e validação para coletar as métricas

    O Google Adwords, avançou para o Google Ads
